### DBMS를 사용하는 이유?
- 기존파일 시스템의 제한
	- 응용 프로그램과 데이터 화일의 종속
	- 일부 데이터 중복
	- 데이터 분리와 이에 따른데이터 고립
	- 비 호환 데이터 파일 형식
	- 이해하기 어려움
- **데이터의 독립성**
	- 효율적인 접근
	- 응용 개발 시간 단축
	- 데이타 무결성
	- 데이터 중복 최소화
	- 보안
	- 동일한 데이타 관리
	- 동시 접근
	- 손상 복구
